# Paper

本目录包含论文写作入口和既有项目 summary。

| 文件或目录 | 用途 |
|---|---|
| `main.tex` | 新论文主文件 |
| `sections/` | 新论文各章节 |
| `tables/` | 手工维护或论文专用表格 |
| `figures/` | 手工维护或论文专用图 |
| `output/` | LaTeX 编译输出，默认不提交 |
| `supply_chain_buyer_power_project_summary.*` | 既有项目路线 summary，保留作参考，不作为新论文入口 |

编译新论文：

```powershell
..\scripts\build_paper.ps1
```
